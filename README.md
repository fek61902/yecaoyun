## 三种网络线路怎么选？

这是很多人纠结的地方，野草云香港VPS有三种带宽类型：

**优质BGP** — 电信回程走CMI，联通回程走CU/NTT/CMI均衡，移动回程走Level3，国际方向择优选路。大陆三网访问都有优化，是大多数建站用户的首选。

**精品BGP** — 电信回程走CU，联通回程走CU，移动回程走CMI。三网回程都走联通骨干为主，对联通用户体验更好，延迟更低。不追求大带宽、注重稳定低延迟的可以考虑。

**国际BGP** — 不含CMI、CU等国内优化路由，线路择优选路。大陆访问延迟偏高，但带宽大（最高500Mbps），适合面向东南亚或海外用户的跨境电商独立站、外贸业务。

简单总结：个人博客/小工具用优质BGP，外贸建站用国际BGP，对延迟极度敏感的用精品BGP。
